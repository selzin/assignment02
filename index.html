GIF Image

  1. An animated GIF is a series of frames,

    and isn't very efficient with file size.

  2. It has low resolution.

  3. It is limited to 256 colors.

  4. It has no sound.

  5. It has huge size.

MP4 Video

  1. An animated MP4 is an compressed

    file format that can contain not

    only video,but also audio and subtitles.

  2. It has large variety of formats.

  3. It allows the viewer to play and

    pause whenever they want.
